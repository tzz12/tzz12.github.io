可以使用filter

``` javascript
todos: [
	{ id: id++, text: 'Learn HTML' },
	{ id: id++, text: 'Learn JavaScript' },
	{ id: id++, text: 'Learn Vue' }
  ]
```

``` javascript
removeTodo(todo) {
  // ...
  this.todos = this.todos.filter(item=>item.id !=todo.id)
}
```
