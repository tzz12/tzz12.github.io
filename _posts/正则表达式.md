基本语法结构
``` javascript
const expression = /pattern/flags;
//等价于
const expression = new RegExp(pattern,flags)
```
pattern 可以是任何的正则表达式
flags 可以拥有0个或者多个，值有以下可选值：

 - ==g #8BC34A== 全局模式，表示查找字符串的 全部内容，而不是找到第一个内容就结束
 - ==i== 不区分大小写
 - ==m== 多行模式
 - ==u== Unicode模式
